# Machine-Car
A machine car control coding repository.
项目目标： 制作一个控制电机与舵机的程序，使得PS2手柄可以通过蓝牙连接来操控小车前进、后退、转弯与机械臂进行操作。
依赖库：PS2X_lib.h，已从外部引入。
编译与烧录：通过Arduino IDE进行。

## 当前控制逻辑
- 左摇杆 Y 轴映射为前后速度（`vx`）。
- 左摇杆 X 轴映射为横移速度（`vy`）。
- 右摇杆 X 轴映射为自旋角速度（`wz`）。
- 麦克纳姆四轮目标速度混控：
  - 前左轮：`vx + vy + wz`
  - 前右轮：`vx - vy - wz`
  - 后左轮：`vx - vy + wz`
  - 后右轮：`vx + vy - wz`
- 当四轮目标速度中任一超限时，会按比例统一缩放；实际电机速度按斜率限制逐步逼近目标速度，并始终受上下限保护。
- 电机与舵机已合并到同一主循环：`loop_ps2()` -> `loop_key()` -> `loop_servo()`。
- 手柄输入统一读取一次：摇杆用于底盘运动；方向键、肩键、功能键用于 6 路舵机控制（`SELECT` 全部回中）。
- 舵机采用 `millis()` 非阻塞平滑更新；当手柄链路异常时执行统一安全策略（电机停转，舵机回中）。
