# Machine-Car
A machine car control coding repository.
项目目标： 制作一个控制电机与舵机的程序，使得PS2手柄可以通过蓝牙连接来操控小车前进、后退、转弯与机械臂进行操作。
依赖库：PS2X_lib.h，已从外部引入。
编译与烧录：通过Arduino IDE进行。

## 当前控制逻辑
- 左摇杆 Y 轴映射为前后速度（`vx`）。
- 左摇杆 X 轴映射为横移速度（`vy`）。
- 右摇杆 X 轴映射为自旋角速度（`wz`）。
- 麦克纳姆四轮目标速度混控：
  - 前左轮：`vx + vy + wz`
  - 前右轮：`vx - vy - wz`
  - 后左轮：`vx - vy + wz`
  - 后右轮：`vx + vy - wz`
- 当四轮目标速度中任一超限时，会按比例统一缩放；实际电机速度按斜率限制逐步逼近目标速度，并始终受上下限保护。
- 新增“驻车抗推模式”：摇杆持续回中后自动进入，任一摇杆明显操作立即退出。
- 驻车抗推使用编码器反馈速度 `g_vehicle_speed` 做零速保持（前后双向抗推），并带最小保持扭矩、最大扭矩限幅与超时降扭保护。
